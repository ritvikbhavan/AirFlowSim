# AirFlowSim

AirFlowSim is a Python-based 3D airflow visualization prototype built with PySide6 and PyVistaQt.

## Iteration 1

This version opens a resizable window, displays a 3D viewport, creates a 100 x 100 x 10 m room, places a centered ceiling fan and two columns, and enables camera rotation, zoom, and pan.

## Run

```bash
python main.py
```

## Dependencies

```bash
pip install -r requirements.txt
```
