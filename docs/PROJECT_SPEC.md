# AirFlowSim

## Objective
Build a lightweight engineering airflow simulator for indoor spaces.

## Technology Stack

- Python 3.12
- PySide6
- PyVista
- PyVistaQt
- NumPy

## Performance Target

- 50,000 particles
- 60 FPS target

## Current Features

- Room visualization
- Ceiling fan visualization
- Particle airflow visualization
- Velocity field
- Multi-fan support

## Future Features

- Streamlines
- Velocity vectors
- Floor interaction
- Column wake effects
- Pressure field
- Temperature field
- Turbulence model
- Room import/export
